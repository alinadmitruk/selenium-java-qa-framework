# Selenium Java QA Automation Framework
## SauceDemo UI Testing Project

##  Project Overview
This is a UI automation testing project built using Selenium WebDriver and Java.
The project covers end-to-end testing of a demo e-commerce application.

Target site: https://www.saucedemo.com/

##  Test Scenarios Covered

- Valid login test
- Invalid login test
- Product list verification
- Add item to cart test
- Cart not empty test
- Remove item from cart test

##  Framework Structure

- Page Object Model (POM)
- BaseTest for WebDriver setup and teardown
- BasePage for reusable driver and wait logic

##  Technologies Used

- Java 21
- Selenium WebDriver
- JUnit 5
- ChromeDriver

##  How to Run Tests

1. Clone repository
2. Open project in IntelliJ IDEA
3. Install dependencies (Maven/Gradle)
4. Run tests from tests package
